## About The Project

This is my 2nd front-end project using React.js. This project creates a to-do list with edit and delete features. 


### Built With

* [![React][React.js]][React-url]


<!-- GETTING STARTED -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/ToDoList.git
   ```
2. Start NPM project
   ```sh
   npm start
   ```
